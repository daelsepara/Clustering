# Data Sets

Data sets for testing GTK clustering software are available on this page. Each line in the text file is a single data point. The values in each line are separated by a delimiter character that is either a tab (\t) character or a comma (,). The last value in each line is the cluster number of that data point.

Citations for these data sets were adopted from:
